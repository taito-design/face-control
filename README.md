あなたは今、キーボードを触っていますか？
マウスに手を伸ばしていますか？

**やめてください。もったいない。**

あなたにはまだ使っていない器官があります。
そう、**顔**です。

---

このアプリは、あなたの顔の動きだけでPCを操作します。

- **口を開ける** → ページが下がる
- **眉を上げる** → ページが上がる
- **右目をウィンク** → 動画が止まる・動く
- **左目をウィンク** → システム自体のON/OFF

料理中でも、手が汚れていても、両手が塞がっていても。
あなたの顔が、マウスになります。

---

使用技術は Python / OpenCV / MediaPipe。
カメラ映像は外部に一切送信されません。
あなたの顔は、あなたのPC内にだけあります。**安心してください。**

動作環境：Windows（Pythonインストール必要）
所要時間：pip install 3分、習得 5分、感動 永遠
